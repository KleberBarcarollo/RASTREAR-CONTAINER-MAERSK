# RASTREAR-CONTAINER-MAERSK
RASTREAR CONTAINER MAERSK
Workflow voltado para rastrear um container baseado em um numero padrão de rastreamento do site https://www.maersk.com/tracking/

Sobre este fluxo de automação:
Este processo realiza a coleta dos status do site da MAERSK, para posterior envia a uma planilha ou banco de dados para posterior analises do time de comercio exterior.

Tipo: Web
Pré-requisitos: Possui os números para tracking.
